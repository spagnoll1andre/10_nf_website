## 4. FONT E COLORI

Riepilogo dei token attivi in `src/app/globals.css` da usare come riferimento durante lo sviluppo.

### Font

| Token CSS | Valore | Uso suggerito |
|---|---|---|
| `--font-sans` | `Inter, sans-serif` | Body text, label, UI |
| `--font-serif` | `Inter Tight, sans-serif` | Titoli display, H1, H2 |
| `--font-mono` | `Inter Tight, sans-serif` | Codice, badge tecnici |

> **Nota:** Il progetto usa esclusivamente Inter e Inter Tight. Salix usa Geist + Inter + Outfit — se si vuole avvicinarsi al registro visivo di Salix, valutare se aggiungere un secondo typeface per i display heading.

|---|---|---|
